Stress-Test Game Overview

You are in a group of {{ C.PLAYERS_PER_GROUP }} players for {{ C.NUM_ROUNDS }} rounds with random rematching each round.

Every round has three decisions:

  1. Send a signal (0-100).
  2. State beliefs about others.
  3. Submit a full conditional contribution schedule (strategy method).

You receive {{ C.ENDOWMENT }} each round. The total contribution is multiplied by {{ C.MULTIPLIER }} and shared equally.

Extra points come from belief accuracy: contribution belief bonus up to {{ C.BELIEF_BONUS_CONTRIB_MAX }}, signal belief bonus up to {{ C.BELIEF_BONUS_SIGNAL_MAX }}.
