How this stress module works

You are placed in groups of {{ C.PLAYERS_PER_GROUP }} and rematched every round for {{ C.NUM_ROUNDS }} rounds.

In each round:

  1. Stage 1: contribution + belief about reaching threshold.
  2. Wait for all group members.
  3. Stage 2: revise your contribution after seeing stage-1 aggregate data.
  4. Wait for all group members, then payoffs are computed.

If final group contribution reaches {{ C.THRESHOLD }}, each player gets coordination bonus {{ C.COORDINATION_BONUS }}.
